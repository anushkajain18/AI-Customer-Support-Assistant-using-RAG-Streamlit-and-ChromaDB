# AI Customer Support Assistant using RAG

## Overview
An AI-powered customer support assistant built using Streamlit, LangChain, and ChromaDB.

## Features
- Retrieval-Augmented Generation (RAG)
- Document-based Question Answering
- Streamlit User Interface
- ChromaDB Vector Database

## Technologies Used
- Python
- Streamlit
- LangChain
- ChromaDB

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
